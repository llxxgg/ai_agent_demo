# 个人简历网站规格文档

## 1. 项目概述
- **项目名称**: Minimal Resume
- **类型**: 单页个人简历网站
- **核心功能**: 展示个人基本信息、技能、项目经验和联系方式
- **目标用户**: 招聘方、合作伙伴

## 2. UI/UX 规格

### 布局结构
- **整体**: 单页垂直滚动布局
- **宽度**: 最大1200px，居中显示
- **区块**:
  - 头部 (Hero): 全屏高度，姓名+职位
  - 关于我: 基本信息模块
  - 技能: 技能展示
  - 项目: 项目作品展示
  - 联系方式: 联系信息
  - 页脚: 版权信息

### 响应式断点
- 桌面: > 1024px
- 平板: 768px - 1024px
- 手机: < 768px

### 视觉设计

#### 配色方案 (黑白灰)
- 主背景: #FFFFFF (纯白)
- 次背景: #F5F5F5 (浅灰)
- 主文字: #1A1A1A (近黑)
- 次文字: #666666 (中灰)
- 强调色: #333333 (深灰)
- 分割线: #E0E0E0 (浅灰)
- 悬停效果: #000000 (纯黑)

#### 字体
- 标题字体: "Cormorant Garamond", serif (优雅衬线)
- 正文字体: "DM Sans", sans-serif (现代无衬线)
- 姓名: 64px (桌面) / 40px (手机)
- 副标题: 24px (桌面) / 18px (手机)
- 正文: 16px
- 小字: 14px

#### 间距系统
- 区块间距: 120px (桌面) / 80px (手机)
- 内容内边距: 80px (桌面) / 24px (手机)
- 元素间距: 24px
- 行高: 1.8

#### 视觉效果
- 卡片阴影: 0 4px 20px rgba(0,0,0,0.05)
- 过渡动画: 0.4s ease
- 滚动淡入: 元素进入视口时淡入上移
- 悬停: 轻微上移 + 阴影加深

### 组件

#### 头部 Hero
- 全屏高度 (100vh)
- 居中显示姓名和职位
- 向下滚动提示箭头 (带动画)

#### 基本信息卡片
- 头像 (圆形，120px)
- 姓名、职位
- 简短介绍 (2-3行)
- 社交链接 (GitHub, LinkedIn, Email 图标)

#### 技能模块
- 技能分类标签
- 技能项: 技能名 + 熟练度条 (灰度渐变)
- 网格布局: 2列 (桌面) / 1列 (手机)

#### 项目模块
- 项目卡片 (3列桌面 / 2列平板 / 1列手机)
- 卡片内容: 项目图片、名称、描述、技术栈、链接
- 悬停效果: 上浮 + 阴影

#### 联系方式
- 居中显示联系方式列表
- 邮箱、电话、位置
- 联系表单 (可选)

#### 页脚
- 简洁版权信息
- 返回顶部按钮

## 3. 功能规格

### 核心功能
1. 响应式布局适配所有设备
2. 平滑滚动导航
3. 滚动动画 (Intersection Observer)
4. 社交链接跳转
5. 项目链接跳转

### 交互行为
- 导航点击: 平滑滚动到对应区块
- 卡片悬停: 上浮 + 阴影效果
- 社交图标悬停: 颜色变化 + 缩放
- 滚动: 触发元素淡入动画

### 动画效果
- Hero 文字: 加载时淡入
- 各区块: 进入视口时从上淡入
- 技能条: 从0渐变到实际宽度
